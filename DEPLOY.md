# Deploy to Vercel

1. Open Vercel and create a new project.
2. Upload this folder as the project root.
3. Add Environment Variables:

   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`

4. Deploy.
5. Open the deployment URL and submit a test form.

The Telegram token must stay in Vercel Environment Variables, not in public files.
